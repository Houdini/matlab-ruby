Class Matlab::CellMatrix
In: lib/matlab/driver/native/conversions.rb
lib/matlab/matrix.rb
Parent: Object

The CellMatrix class is used for storing values that are sent to or received from MATLAB cell matrices. Usage: 

  require 'matlab'

  cell_matrix = Matlab::CellMatrix.new(20, 400)
  20.times { |m| 400.times { |n| cell_matrix[m, n] = rand.to_s } }
